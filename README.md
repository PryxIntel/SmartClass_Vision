# 🎓 SmartClass Vision

An **AI-Powered Face Recognition Attendance Management System** built using **YOLOv8**, **DeepFace**, **OpenCV**, and **SQLite**.

The project automates student identification and attendance tracking, eliminating manual attendance processes through Computer Vision and Facial Recognition technologies.

---

## 🚀 Features

- **Real-Time Face Detection** using YOLOv8
- **Facial Recognition** powered by DeepFace and FaceNet512
- **Automated Attendance Tracking**
- **Student Registration & Profile Management**
- **Attendance Report Generation**
- **SQLite Database Integration**
- **Interactive GUI** built with CustomTkinter

---

## 🛠️ Tech Stack

- **Programming Language:** Python
- **Computer Vision:** OpenCV, YOLOv8
- **Face Recognition:** DeepFace, FaceNet512
- **Database:** SQLite
- **GUI Framework:** CustomTkinter

---

## 🔧 Core Modules

- **Face Detection** – Detects faces from live camera feeds
- **Face Recognition** – Identifies registered students
- **Attendance Management** – Automatically records attendance
- **Student Registration** – Stores facial embeddings and student profiles
- **Database Management** – Maintains attendance records and student data
